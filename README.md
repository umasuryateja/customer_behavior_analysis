# 📊 Customer Behavior Data Analysis Project

## 📌 Overview

This project demonstrates a complete end-to-end data analytics workflow, covering data processing, analysis, and visualization. The goal is to analyze customer behavior data to extract meaningful insights that can support business decision-making.

The project includes data cleaning, exploratory data analysis (EDA), SQL-based analysis, and an interactive Power BI dashboard to present key findings. Customer behavior analysis helps businesses understand purchasing patterns and improve strategies for retention and sales growth .

---

## 📂 Dataset

* The dataset contains customer-related information such as demographics, purchasing behavior, and transaction details
* Includes structured data suitable for analysis in Python and SQL
* Used for identifying trends, patterns, and customer segments

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI** (Dashboard & Visualization)
* **Jupyter Notebook** (Analysis)
* **Gamma AI** (Presentation Creation)

## ⚙️ Project Steps

### 1. Data Loading & Preparation

* Imported dataset using Python
* Checked data types, missing values, and duplicates

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized and formatted data

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Identified purchasing trends
* Visualized patterns using charts

### 4. SQL Analysis

* Loaded cleaned data into SQL database
* Executed queries to analyze:

  * Customer segments
  * Purchase patterns
  * Business KPIs

### 5. Data Visualization

* Built interactive dashboards in Power BI
* Displayed key metrics such as:

  * Sales trends
  * Customer insights
  * Category performance

### 6. Reporting & Presentation

* Created a structured project report
* Designed presentation slides using Gamma AI
